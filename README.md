# kurs
## install
```
$ go get github.com/ridhoadya/kurs
$ go build kurs.go
```

## usage
input parameter with 3 digit of currency
```
$ ./kurs USD
Kurs Terhadap: USD
Nilai: 1.00
Kurs Jual: Rp. 14,187.00
Kurs Beli: Rp. 14,045.00
```
